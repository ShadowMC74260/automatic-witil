Custom Office Installation

1. Office Configurator

Configure Office as you like here, and export the file as "Officeconfig.xml" to any drive, and then copy it to C:

If you want latest office Select "Microsoft 365 Apps for Enterprise (Select teams or no Teams based on preference)

2. Office Deployment Tool

Download the file, and save it anywhere. Run the file, and select the directory to folder called "Office Setup" on any drive.

Copy the setup.exe into the C:

If the link does not work, simply search "ODT" on your browser.

3. Open Command Prompt and run the following command:

cd /d C:\

setup.exe /configure Officeconfig.xml

If you are unable to get the command to work, you can try "open terminal here" in the right-click menu and use this command in Powershell instead:

./setup.exe /configure Officeconfig.xml

4. Let office install, and then activate through above method.
